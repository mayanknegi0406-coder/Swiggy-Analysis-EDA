# 🍽️ Swiggy Analysis EDA (Python)

This project performs **Exploratory Data Analysis (EDA)** on Swiggy dataset to uncover insights related to restaurants, ratings, pricing, cuisines, and customer preferences.

## 🔍 Key Insights
- Top-rated restaurants and popular cuisines
- Price distribution across restaurants
- Rating trends and customer preferences
- City/area-wise restaurant availability
- Relationship between pricing and ratings

## 📈 Features
- Data cleaning and preprocessing
- Univariate and bivariate analysis
- Visualizations using charts and graphs
- Restaurant and cuisine trend analysis
- Actionable business insights

## 🛠 Tools Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

## 🎯 Purpose
The goal of this project is to analyze food delivery market trends, customer preferences, and restaurant performance while showcasing Python data analytics skills.
